VibeDispatcher (VS Code + Mobile Beacon) Claude Code Jarvis V (Claude Code CLI) Shared / synergy zone

Tier 1High value, low effort — do these first

  1. Extract KeychainHelper + HallucinationDetector into a shared SPM package (~1 day) — both products depend on it; closes VD's flat-file-API-key security finding (Regression Report C2)
  2. CCJV publishes voice events to VD's relay → mobile Beacon surfaces CCJV transcripts (~1-2 days) — CCJV gets away/mobile read for free

Tier 2Medium value, medium effort

  1. Shared VoiceCatalog SPM package — single source of truth for voice/backend definitions (~3-5 days)
  2. VD's TerminalMonitor enriched with CCJV's badge-parsing for unified non-VS-Code agent surfacing (~2 days)

Tier 3High value, big build

  1. Shared NSPanel HUD framework + brand kit + intent router (~1-2 weeks) — ties both products together visually + functionally; positions for the third menu-bar app you'll inevitably ship
Anti-pattern to avoid: merging the two products. Different surface targets (VS Code vs Claude Code CLI), different mental models (alert-triage vs character-driven voice). The win is shared substrate + cross-publishing, not consolidation.
